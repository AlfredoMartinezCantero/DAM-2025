# Conclusión:
Gracias a este ejercicio se consiguen varios objetivos importantes. En primer lugar, se pone en práctica el uso de HTML y CSS para crear una página web estructurada y visualmente atractiva. Además, se aprende a organizar el contenido de manera efectiva, utilizando elementos semánticos como encabezados, párrafos y enlaces. Por último, se fomenta la creatividad al diseñar un portafolio que refleja el estilo personal del autor.
Un error común al crear el código de html sería, por ejemplo, olvidar cerrar las etiquetas correctamente, lo que puede llevar a problemas de visualización en los navegadores.
