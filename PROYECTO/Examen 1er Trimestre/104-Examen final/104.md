# Introdución:
Este ejercicio consiste en unir los exámenes de Bases de datos, Lenguaje de marcas y Programación para crear una aplicación web que muestre un portafolio de piezas de arte almacenadas previamente en una base de datos en MySQL llamada portafolioexamen.

El objetivo principal es mostrar cómo se pueden integrar las diferentes tecnologías para crear una aplicación web dinámica y funcional.

---
```
import mysql.connector          # Importo el conector a base de datos
from flask import Flask         # Importo 

db = mysql.connector.connect(
    host="localhost",
    user="usuario",  
    password="contraseña",
    database="portafolioexamen "
)

cursor = db.cursor()            # Creo el cursor para hacer consultas
app = Flask(__name__)            # Creo la aplicación Flask

@app.route("/")                  # Defino el endpoint principal
def holamundo():
    cursor.execute("SELECT * FROM vista_portafolio")  # Ejecuto la consulta        
    filas = cursor.fetchall()      # Guardo el resultado en una lista

########## AQUI PONGO EL INICIO HASTA EL MAIN 
    cadena = '''

if __name__ == "__main__":                # Si este es el archivo principal
    app.run(debug=True)                   # Ejecuta la web
```

# Conclusión:
Así demostramos cómo combinar diferentes tecnologías para crear uuna aplicación web dinámica que interactúa con una base de datos.

Utilizando Flask para el backed y MySQL para el almacenamiento de datos logrando pues desarrollar una plataforma sencilla para mostrar un portafolio de piezas de arte.

Un error que puede surgir al escribir el código puede ser el no cerrar bien las etiquetas del HTML u olvidarse de usar el `@app.route("/")`
