# Introducción:
En este ejercicio he aprendido a generar contenido web dinámico y estructurado utilizando Flask y Python, el objetivo ha sido crear una tabla HTML que represente 12 semanas con la estructura de 1 a 31 días, para lograr esto, he combinado la lógica de programación con el lenguaje de marcas utilizando un bucle `for` externo para iterar 12 veces, asi representando las filas de la tabla, y un bucle anidado para generar los números del 1 al 31.

---
```
from flask import Flask

app = Flask(__name__)

@app.route("/")
def tabla_dinamica():
    # Inicializo la cadena con la estructura básica del HTML y CSS
    cadena = """
    <!DOCTYPE html>
    <html lang="es">
    <head>
        <meta charset="UTF-8">
        <title>Calendario Dinámico</title>
        <style>
            body { font-family: Arial, sans-serif; background-color: #f0f0f0; padding: 20px; }
            table { width: 100%; border-collapse: collapse; background-color: white; }
            th, td { border: 1px solid #ddd; padding: 10px; text-align: left; }
            th { background-color: #4CAF50; color: white; }
            tr:nth-child(even) { background-color: #f2f2f2; }
            .dias { color: #555; font-size: 0.9em; }
        </style>
    </head>
    <body>
        <h1>Tabla de 12 Semanas (Días 1-31)</h1>
        <table>
            <thead>
                <tr>
                    <th>Semana / Bloque</th>
                    <th>Días</th>
                </tr>
            </thead>
            <tbody>
    """

    # Genera las 12 filas (Semanas)
    # range(1, 13) genera números del 1 al 12
    for semana in range(1, 13):
        cadena += f"<tr>"
        cadena += f"<td><strong>Semana {semana}</strong></td>"
        cadena += f"<td class='dias'>"
        
        # Genera los días del 1 al 31 para cada semana
        # range(1, 32) genera números del 1 al 31
        for dia in range(1, 32):
            cadena += f"{dia} "
            
            # Añado un separador visual (guion) si no es el último día
            if dia < 31:
                cadena += "- "
        
        cadena += "</td>"
        cadena += "</tr>"

    # Cierro las etiquetas HTML
    cadena += """
            </tbody>
        </table>
    </body>
    </html>
    """
    
    return cadena

if __name__ == "__main__":
    app.run(debug=True)
```

# Conclusión:
Con este código he aprendido como incrustar la lógica de Python dentro de las vistas web, comprobando que el uso de bucles es fundamental cuando necesitamos representar datos de filas y columnas.

Un error común al generar HTML desde Python es olvidar cerrar correctamente las etiquetas dentro o fuera del bucle, por ejemplo, en las primeras versiones del código coloqué el cierre de la etiqueta `</tr>` fuera del bucle `for` de las semanas, lo que pasó fue que la tabla se rompió visualmente, ya que el navegador no sabía dónde termina cada fila.
