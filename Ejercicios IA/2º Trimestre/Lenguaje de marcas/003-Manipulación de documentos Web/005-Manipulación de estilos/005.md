# Introducción:
En este ejercicio se conecta la lógica de validación de datos con la respuesta visual del navegador, he comenzado seleccionando el elemento `input` usando `document.querySelector`, lo que me permite "atrapar" el objeto HTML para manipularlo desde JavaScript, para detectar la interacción del usuario en tiempo real he asignado una función anónima al evento `onkeyup`, que se dispara cada vez que levanto el dedo de una tecla, dentro de esta lógica utilizo `this.value` para capturar el contenido actual del campo y `this.value.length` para evaluar cuántos caracteres se han introducido, la gestión visual la realizo de forma dinámica, es decir, en lugar de cambiar estilos en línea, utilizo los métodos `classList.add` y `classList.remove` para alternar entre verde y rojo, basándome en una condición `if/else` (que la longitud sea exactamente 9), proporcionando así un feedback visual inmediato.

---
```
<!doctype html>
<html lang="es">
  <head>
    <meta charset="utf-8">
    <title>Validación Visual Condicional</title>
    <style>
      /* Defino las clases visuales en CSS */
      .rojo{
          background:rgb(255,200,200); /* Rojo para el error */
          border: 1px solid red;
      } 
      .verde{
          background:rgb(200,255,200); /* Color para el exito */
          border: 1px solid green;
      } 
      body{ font-family: sans-serif; padding: 20px; }
    </style>
  </head>
  <body>
    
    <h3>Validación de DNI/Teléfono (Ejemplo)</h3>
    <p>Esta entrada debe tener 9 caracteres exactamente para ser válida.</p>
    
    <input type="text" class="rojo" placeholder="Escribe aquí...">

    <script>
      // Selecciono la primera etiqueta input del documento
      let entrada = document.querySelector("input"); 

      // Asigno el evento
      entrada.onkeyup = function() { 
        
        let valor = this.value;       // Obtengo el texto escrito
        let longitud = valor.length;  // Calculo cuántos caracteres tiene

        // Lógica de validación
        if (longitud == 9) { 
          // Si es correcto (9 caracteres)
          this.classList.add("verde");    // Añado estilo de éxito
          this.classList.remove("rojo");  // Quito estilo de error
        } else { 
          // Si es distinto de 9
          this.classList.remove("verde"); // Quito estilo de éxito
          this.classList.add("rojo");     // Añado estilo de error
        }
      };
    </script>
  </body>
</html>
```

# Conclusión:
Con este ejercicio aprendemos la base de la validación de formularios en el Frontend, comprendiendo que no es buena práctica modificar los estilos CSS directamente desde JavaScript como por ejemplo `style.background = 'red'`, sino que es mucho más limpio,profesional y mantenible añadir y quitar clases, separando así el CSS de JS.
