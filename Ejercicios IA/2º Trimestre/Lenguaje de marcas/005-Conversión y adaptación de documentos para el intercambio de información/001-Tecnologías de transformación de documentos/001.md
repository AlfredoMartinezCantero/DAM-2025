# Introducción:
En este ejercicio he puesto en práctica la generación automática de documentos PDF a partir de texto en Python, para ello, he hecho uso de la librería `fpdf`, creando una función llamada `text_to_pdf`, el proceso comienza instanciando la clase principal `FPDF()` y preparando el documento con `add_page()` y `set_font()` para definir el estilo visual, la parte central del ejercicio consiste en procesar el texto de entrada utilizando el método `.split("\n")`, dividiendo la cadena en líneas individuales para recorrerlas con un bucle, por último utilizo el método `pdf.cell()` para escribir cada línea en el documento, asegurando el salto de línea automático con `ln=True` antes de guardar el resultado en disco usando `pdf.output()`.

---
```
from fpdf import FPDF

def text_to_pdf(text, filename="salida.pdf"):
    pdf = FPDF()
    pdf.add_page()
    pdf.set_font("Arial", size=12)

    for line in text.split("\n"):
        pdf.cell(0, 10, txt=line, ln=True)

    pdf.output(filename)

text_to_pdf("Hola este es un texto desde Python")
```
