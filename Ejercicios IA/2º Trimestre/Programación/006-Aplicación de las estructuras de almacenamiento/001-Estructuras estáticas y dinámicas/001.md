# Introducción:
Este programa nos ayuda a comprender las estructuras de datos básicas, específicamente las listas. El objetivo es crear una colección ordenada de cadenas de texto, verificar su contenido, acceder a elementos específicos mediante su índice posicional y modificar datos existentes.

`lista_canciones` Inicializa la lista reservando espacio en memoria para elementos de tipo string. Python asigna automáticamente índices numéricos a cada elemento, comenzando siempre por el 0, por eso al poner `print(lista_canciones[0])` recuperamos el primer valor, en este caso la canción "Desafiante".

`lista_canciones[1] = "La ciudad de los sueños"` Esto ejecuta una sobrescritura del valor seleccionado, el programa localiza el espacio asociado al índice 1 y reemplaza el valor anterior con un nuevo dato.

---
```
lista_canciones = ["Desafíate",
                   "Eres un lobo",
                   "La muerte es solo el final del comienzo"
                   ]

print(lista_canciones)

print(lista_canciones[0])

lista_canciones[1] = "La ciudad de los sueños"

print(lista_canciones)
```
---

# Conclusión:

Este ejercicio demuestra la flexibilidad de las listas para gestionar colecciones de datos dinámicos. A diferencia de las tuplas, las listas permiten la edición de sus elementos en tiempo de ejecución, lo cuál es muy útil para programas que requieran actualizar la información de manera recurrente, como un carrito de la compra o una lista de reproducción de música.

Un error común al escribir este programa sería por ejemplo intentar acceder a `lista_canciones[3]` pensando que es el tercer elemento, cuando en realidad el valor 3 corresponde a una cuarta posición que, en este caso, no existe.
