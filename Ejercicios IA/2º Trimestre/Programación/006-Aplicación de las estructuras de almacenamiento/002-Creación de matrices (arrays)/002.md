# Introducción:
Este ejercicio enseña a trabajar con listas dinámicas y a utilizar la serialización de datos en Python mediante la libreria `pickle`.

El objetivo del ejercicio es crear un sencillo sistema de gestión de comidas que permita añadir, listar, guardar y cargar elementos, almacenando la información en un archivo binario (datos.bin), para ello, utilizamos una lista estándar llamada `menu` que actuará como contenedor temporal de la información durante la ejecución del programa.

La libreria `pickle` se emplea para dos funciones clave: `dump` (para escribir la lista en el disco duro) y `load` (para recuperarla), garantizando que los datos introducidos por el usuario persistan entre diferentes sesiones.

Además, se implementa un bucle infinito `while True` que ofrece un menú interactivo para manipular la lista según las necesidades del usuario.

---
```
import pickle

#Creamos la lista vacía
menu = []

print("--- GESTOR DE COMIDAS ---")

while True:
    print("\nELIGE UNA OPCIÓN:")
    print("1. Anadir comida")
    print("2. Listar comidas")
    print("3. Guardar lista en archivo")
    print("4. Cargar lista desde archivo")
    print("5. Salir")
    
    opcion = input("\nTu opcion: ")

    if opcion == "1":
        # Añadimos una nueva comida
        nueva_comida = input("Introduce el nombre de la comida: ")
        menu.append(nueva_comida)
        print("OK: '" + nueva_comida + "' añadida al menú.")

    elif opcion == "2":
        # Listamos las comidas
        print("\n--- LISTADO ACTUAL ---")
        if len(menu) == 0:
            print("(La lista esta vacia)")
        else:
            # Usamos enumerate para tener el indice (i) y el valor (comida)
            for i, comida in enumerate(menu):
                # Sumamos 1 al indice porque empieza en 0
                numero = i + 1
                # Convertimos 'numero' a string con str() para poder sumarlo al texto
                print(str(numero) + ".- " + comida)
        print("----------------------")

    elif opcion == "3":
        # Guardamos en binario
        try:
            archivo = open("datos.bin", "wb")
            pickle.dump(menu, archivo)
            archivo.close()
            print("ÉXITO: Lista guardada correctamente en 'datos.bin'.")
        except:
            print("ERROR: No se pudo guardar el archivo.")

    elif opcion == "4":
        # Cargar desde binario
        try:
            archivo = open("datos.bin", "rb")
            menu = pickle.load(archivo)
            archivo.close()
            print("EXITO: Datos cargados.")
        except:
            print("ERROR: No existe el archivo o hubo un error al leerlo.")

    elif opcion == "5":
        print("Saliendo del programa...")
        break

    else:
        print("Opcion no valida.")
```

# Conclusión:
Este ejercicio nos ayuda a entender a cómo combinar el uso de estructuras de datos básicas, como las listas, con el manejo de archivos binarios para lograr la persistencia de la información.

Gracias a la simplicidad del módulo pickle, es posible almacenar el estado completo de la lista menu en el disco duro sin necesidad de procesar el texto manualmente. Esto permite que el programa recupere la información tal y como estaba en ejecuciones anteriores, cumpliendo con el objetivo fundamental de cualquier sistema de gestión de datos.
