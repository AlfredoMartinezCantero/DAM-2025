# Introducción:
Este ejercicio enseña a como estructurar datos utilizando arrays asociativos, a diferencia de los arrays numéricos, se asignan claves descriptivas ('nombre', 'genero') que dan contexto semántico al dato.

El paso clave ha sido utilizar la función `json_encode()`, esta función actúa como un "traductor", serializando la estructura de datos interna de PHP a una cadena de texto bajo el estándar JSON, que es el formato universal para enviar datos entre servidores y clientes.

---
```
<?php
    $musica = [];
    $musica['nombre'] = "Extremoduro";
    $musica['genero'] = "Rock";
    $musica['artista'] = "Robe";

    $jsonMusica = json_encode($musica);
    echo $jsonMusica;
?>
```
---
# Conclusión:
El objetivo de este ejercicio es hacer ver que no se puede hacer un `echo` directo de un array en PHP, y que `json_encode` es la herramienta obligatoria para transformar esos datos en un string legible que cualquier sistema pueda entender.

Un error común es confundir las variables de entrada y salida. En mi primer intento, escribí `json_encode($jsonMusica)`, intentando codificar la variable destino antes de que existiera, lo que genera un error, `"Undefined variable"`. Es decir, para la función `json_encode` siempre debo darle el array con datos `($musica)` y recoger el resultado en una variable nueva.
