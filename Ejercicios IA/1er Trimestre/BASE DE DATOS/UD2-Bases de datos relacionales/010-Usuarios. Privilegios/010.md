# Introducción:
Este ejercicio busca administrar usuarios en un sistema de BBDD en MySQL, tiene como objetivo listar los usuarios existentes, crear un nuevo usuario y asignarle privilegios específicos sobre una base de datos.
Esto permite gestionar de manera segura quién puede acceder y qué acciones pueden realizar en la base de datos.

---
**-- Mostrar usuarios en el sistema.**
```
SELECT USER, HOST FROM mysql.user;
```
**-- Crear un nuevo usuario.**
```
CREATE USER 'tunombredeusuario'@'tuservidor' IDENTIFIED BY 'tucontraseña';

GRANT USAGE ON *.* TO 'tunombredeusuario'@'tuservidor';

GRANT ALL PRIVILEGES ON empresadam.* TO 'tunombredeusuario'@'tuservidor';

FLUSH PRIVILEGES;
```
---

# Conclusión:
Gracias a este ejercicio se aprende a como otorgar los privilegios necesarios sobre la BBDD y verificar que los cambios se aplican correctamente asi como controlar el acceso y garantizar la seguridad de los datos almacenados en la BBDD.
