# Introducción:
Este ejercicio realiza operaciones básicas en MySQL para familiarizarse con la inserción de datos, el manejo de valores NULL en una base de datos y la manipulación de tablas.

En este ejercicio se trabaja con la BBDD llamada `empresadam` y la tabla `pedidos`, donde se insertan registros y se observa el comportamiento de los valores NULL.

---
**-- Conexión a MySQL.**
```
sudo mysql -u root -p
```
**-- Mostrar bases de datos.**
```
SHOW DATABASES;
```
**-- Seleccionar BBDD empresadam.**
```
USE empresadam;
```
**-- Mostrar tablas en la BBDD.**
```
SHOW TABLES;
```
**-- Ejercicio con la tabla pedidos.**
```
INSERT INTO pedidos (numerodepedido, cliente, producto) 
VALUES ('P001', 'Juan Pérez', 'Laptop');

SELECT * FROM pedidos;
```
**-- Prueba con valores NULL.**
```
INSERT INTO pedidos (numerodepedido, cliente)
VALUES ('P002', 'Ana González');

SELECT * FROM pedidos WHERE numerodepedido = 'P002';
```
---

# Conclusión:
El ejercicio ayuda a entender como se aplica la inserción de datos en una tabla MySQL y a manejar valores NULL. Se puede observar que al no especificar un valor para la columna `producto`, esta queda con un valor NULL, lo que indica la ausencia de datos. Este comportamiento es fundamental para comprender cómo funcionan las bases de datos relacionales y cómo se pueden gestionar los datos incompletos o desconocidos.
