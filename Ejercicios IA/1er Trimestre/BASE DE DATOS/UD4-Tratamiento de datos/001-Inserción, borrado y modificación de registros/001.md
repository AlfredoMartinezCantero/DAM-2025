# Introducción:
Este ejercicio enseña a crear y gesionar una tabla en una base de datos SQL, realizando operaciones básicas como inserción, actualización y eliminación de registros.

`CREATE TABLE clientes`: Crea una tabla llamada "clientes" con varios campos para almacenar información de los clientes.
`INSERT INTO clientes VALUES(...)`: Inserta un nuevo registro en la tabla "clientes" con los datos proporcionados.
`UPDATE clientes SET ... WHERE ...`: Actualiza el nombre de un cliente específico basado en su número de teléfono.
`DELETE FROM clientes`: Elimina todos los registros de la tabla "clientes".

---
**-- Crea una tabla de clientes.**
```
CREATE TABLE clientes (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(50) NOT NULL,
    apellidos VARCHAR(255) NOT NULL,
    telefono VARCHAR(20) NOT NULL,
    email VARCHAR(255) NOT NULL,
    localidad VARCHAR(50) NOT NULL
);
```
**-- Inserta un nuevo cliente.**
```
INSERT INTO clientes VALUES(
    NULL,
    'Rafa',
    'Cantero Trullenque',
    '682601446',
    'rafa.cantero@gmail.com',
    'Valencia'
);
```
**-- Actualizar un registro con WHERE.**
```
UPDATE clientes SET nombre = 'Jose Vicente' WHERE telefono = '620891718';
```

**-- Eliminar todos los registros de la tabla clientes.**
```
DELETE FROM clientes;
```

Asi se demuestra como manejar una tabla de clientes de manera segura y eficiente, aplicando comandos SQL esenciales para trabajar con BBDD.
