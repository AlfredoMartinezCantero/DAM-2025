# Introducción:
Este ejercicio consiste en crear un programa en Python que gestione una agenda de contactos utilizando archivos de texto. 

El programa permitirá al usuario introducir nuevos contactos y leer los contactos existentes desde un archivo llamado `agenda.txt` y cada contacto estará compuesto por un nombre y un email, separados por una coma.

`def nuevo_contacto():` define una función que solicita al usuario el nombre y el email del contacto, y luego escribe esta información en el archivo `agenda.txt`.

`archivo = open('agenda.txt', 'a')` sirve para abrir el archivo en modo de **adición** (`'a'`), lo que significa que los nuevos contactos se agregarán al final del archivo sin borrar los existentes.

`archivo.write(nombre + "," + email + "\n")` escribe el nombre y el email del contacto en el archivo, separados por una coma y seguidos de un salto de línea.

`def leer_contactos():` define una función que lee todos los contactos almacenados en el archivo `agenda.txt` y los muestra en la consola.

`archivo = open('agenda.txt', 'r')` abre el archivo en modo de lectura para poder **leer** (`'r'`) su contenido.

Por ultimo, el programa presenta un menú al usuario usando `if` y `elif` para permitirle elegir entre introducir un nuevo contacto o leer todos los contactos existentes.

---
**Paso 1: Introducir un nuevo contacto**
```
def nuevo_contacto():
    nombre = input("Introduce el nombre del contacto: ")
    email = input("Introduce el email del contacto: ")

    archivo = open('agenda.txt', 'a')
    archivo.write(nombre + "," + email + "\n")
    archivo.close()
```
---
**Paso 2: Leer todos los contactos**
```
def leer_contactos():
    archivo = open('agenda.txt', 'r')
    lineas = archivo.readlines()
    archivo.close()

    for linea in lineas:
        partes = linea.strip().split(",")
        if len(partes) == 2:
            nombre, email = partes
            print("Nombre: " + nombre)
            print("Email: " + email)
            print("----")
```
---
**Paso 3: Interfaz de usuario**
```
print("Menú:")
print("1.- Introducir un nuevo contacto")
print("2.- Leer todos los contactos")
opcion = input("Selecciona una opción: ")

if opcion == "1":
    nuevo_contacto()
elif opcion == "2":
    leer_contactos()
else:
    print("Opción no válida.")
```

# Conclusión:
Este programa ayuda a entender cómo manejar archivos de texto en Python para almacenar y recuperar información, utilizando funciones para organizar el código y una interfaz básica para interactuar con el usuario.
Esto es especialmente útil para aprender sobre flujos de datos, tipos de bytes y caracteres, y las clases relacionadas con la gestión de archivos en Python.
Un error común que se puede cometer sería el no cerrar el archivo después de escribir o leer, lo que puede llevar a pérdida de datos o errores en el programa.
