# Introducción:
Este ejercicio sirve para practicar la creación de interfaces gráficas utilizando la biblioteca Tkinter en Python. El objetivo es crear una ventana que permita al usuario ingresar dos números, y al presionar un botón y, que a continuación, se muestre la suma de esos números en la misma ventana.
Para ello se hace uso de etiquetas, campos de entrada y botones.

`etiqueta1 = tk.Label(ventana, text="Número 1:")` sirve para crear una etiqueta que indica al usuario que debe ingresar el primer número.

`entrada1 = tk.Entry(ventana)` crea un campo de entrada donde el usuario puede escribir el primer número.

Se usa el `def calcular_suma():` para definir una función que se ejecutará cuando el usuario presione el botón. Esta función obtiene los valores ingresados, los convierte a números, calcula la suma y actualiza la etiqueta de resultado.

`except ValueError:` maneja el caso en que el usuario ingrese valores no numéricos, mostrando un mensaje de error en lugar de intentar realizar la suma.

`boton_calcular = tk.Button(ventana, text="Calcular Suma", command=calcular_suma)` crea un botón que, al ser presionado, ejecutará la función `calcular_suma`.

---

```
import tkinter as tk

ventana = tk.Tk()

ventana.title("Suma de dos números")
ventana.geometry("350x150")

etiqueta1 = tk.Label(ventana, text="Número 1:")
etiqueta1.pack()
entrada1 = tk.Entry(ventana)
entrada1.pack()

etiqueta2 = tk.Label(ventana, text="Número 2:")
etiqueta2.pack()
entrada2 = tk.Entry(ventana)
entrada2.pack()

def calcular_suma():
    try:
        num1 = float(entrada1.get())
        num2 = float(entrada2.get())
        resultado = num1 + num2
        etiqueta_resultado.config(text="Resultado: " + str(resultado))
    except ValueError:
        etiqueta_resultado.config(text="Error: ingrese solo números")

boton_calcular = tk.Button(ventana, text="Calcular Suma", command=calcular_suma)
boton_calcular.pack(pady=10)

etiqueta_resultado = tk.Label(ventana, text="Resultado: ")
etiqueta_resultado.pack()
ventana.mainloop()
```
