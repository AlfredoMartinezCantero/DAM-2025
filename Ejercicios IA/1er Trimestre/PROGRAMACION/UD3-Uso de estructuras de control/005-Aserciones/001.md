# Introducción:
En este ejercicio vamos a ver el uso del `assert`, que sirve para verificar si una condición es verdadera, para eso empezamos con el `try`, para que el programa lea el código que puede fallar, al definirle al programa que; `edad = 47` assert "mirará" si la edad que el tiene asignada es la correcta, en este caso, lee que la edad es `48`, por lo que no es igual que la variable previamente definida y falla, así que gracias al `except` el programa es capaz de representar el fallo, y con el `print` podemos crear el mensaje de `Error determinado`.

---

```
edad = 47
try:
    assert edad == 48, "La edad  no es igual a 48"
except:
    print("Error determinado.")
```
---
# Conclusión:
Un error común sería no poner un mensaje en la aserción, lo que haría que el error sea menos claro si ocurriera.
