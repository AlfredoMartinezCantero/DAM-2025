# Introducción:
En este ejercicio se crea una página web personal utilizando HTML, que incluye metadatos, estructura semántica y enlaces a redes sociales, aplicando etiquetas adecuadas para mejorar la accesibilidad y el SEO.

Un ejemplo claro de esto es en el apartado de metadatos, donde se incluyen etiquetas como `<meta name="description" content="Web oficial de Alfredo Martínez Cantero, estudiante de Desarrollo de Aplicaciones Multiplataforma">` para describir el contenido de la página, y `<meta name="keywords" content="Alfredo Martínez Cantero, Python, HTML, C, CSS, C++">` para mejorar la optimización en motores de búsqueda. Además, se utilizan etiquetas semánticas como `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, y `<footer>` para estructurar el contenido de manera lógica y accesible.

# Conclusión:
Este ejercicio ha permitido aplicar conocimientos de HTML para crear una página web personal bien estructurada y optimizada. La inclusión de metadatos relevantes y el uso de etiquetas semánticas no solo mejoran la accesibilidad del sitio, sino que también contribuyen a un mejor posicionamiento en motores de búsqueda. Además, la integración de enlaces a redes sociales facilita la conexión con otros profesionales y la difusión de proyectos personales.

Un error que podría ocurrir al escribir mal el HTML es olvidar cerrar una etiqueta, lo que puede causar que el navegador no interprete correctamente el contenido de la página. Por ejemplo, si se olvida cerrar una etiqueta `<div>`, todo el contenido posterior podría verse afectado, resultando en un diseño desordenado o en la pérdida de funcionalidad de ciertos elementos. Es crucial revisar y validar el código HTML para asegurar su correcta estructura y funcionamiento.
