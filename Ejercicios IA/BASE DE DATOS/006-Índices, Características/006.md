# Introducción:
Este ejercicio consiste en crear una base de datos SQLite para gestionar una lista de clientes. La base de datos debe incluir una tabla llamada "clientes" con las siguientes características:

- Un campo "Identificador" que sea la clave primaria y se autoincremente.

- Un campo "nombre" que no permita valores nulos.

- Un campo "apellidos" que no permita valores nulos.

- Un campo "email" que no permita valores nulos y sea único.
---
Además, se debe implementar un script en Python siguiendo las reglas del CRUD (Crear, Leer, Actualizar, Eliminar) para gestionar los registros de la tabla "clientes".

`def create_connection(db_file):` sirve para crear una conexión a la base de datos SQLite especificada por `db_file`. Si la conexión es exitosa, devuelve el objeto de conexión; de lo contrario, imprime el error y devuelve `None`.

`except Error as e:` captura cualquier excepción que ocurra durante la conexión a la base de datos y la imprime para ayudar en la depuración.

`sql_create_clients_table` contiene la sentencia SQL para crear la tabla "clientes" con los campos especificados.

`c = conn.cursor()` crea un cursor para ejecutar comandos SQL en la base de datos y `c.execute(sql_create_clients_table)` ejecuta la sentencia SQL para crear la tabla.

`def create_client(conn, client):` define una función para insertar un nuevo cliente en la tabla "clientes". Toma como parámetros la conexión a la base de datos y una tupla con los datos del cliente.

`def list_clients(conn):` define una función para listar todos los clientes en la tabla "clientes". Toma como parámetro la conexión a la base de datos.

`def update_client(conn, client):` define una función para actualizar los datos de un cliente existente en la tabla "clientes". Toma como parámetros la conexión a la base de datos y una tupla con los nuevos datos del cliente, incluyendo su identificador.

`def delete_client(conn, id):` define una función para eliminar un cliente de la tabla "clientes" basado en su identificador. Toma como parámetros la conexión a la base de datos y el identificador del cliente a eliminar.

`def main():` es la función principal que gestiona la interacción con el usuario. Crea la conexión a la base de datos y crea la tabla si no existe y presenta un menú para que el usuario pueda elegir entre las diferentes operaciones CRUD.

Por último, el bloque `if __name__ == '__main__':` asegura que la función `main()` se ejecute solo cuando el script se ejecuta directamente, no cuando se importa como un módulo.

**-- Crear la Tabla.**
```
CREATE TABLE clientes (
    Identificador INTEGER PRIMARY KEY AUTOINCREMENT,
    nombre TEXT NOT NULL,
    apellidos TEXT NOT NULL,
    email TEXT NOT NULL UNIQUE
);
```
---
**Script Python para gestionar clientes en SQLite**
```
import sqlite3
from sqlite3 import Error

DB_NAME = 'empresa.db'
def create_connection(db_file):
    conn = None
    try:
        conn = sqlite3.connect(db_file)
    except Error as e:
        print(e)
    return conn
def create_table(conn):
    try:
        sql_create_clients_table = """ CREATE TABLE IF NOT EXISTS clientes (
                                        Identificador INTEGER PRIMARY KEY AUTOINCREMENT,
                                        nombre TEXT NOT NULL,
                                        apellidos TEXT NOT NULL,
                                        email TEXT NOT NULL UNIQUE
                                    ); """
        c = conn.cursor()
        c.execute(sql_create_clients_table)
    except Error as e:
        print(e)
def create_client(conn, client):
    sql = ''' INSERT INTO clientes(nombre,apellidos,email)
              VALUES(?,?,?) '''
    cur = conn.cursor()
    cur.execute(sql, client)
    conn.commit()
    return cur.lastrowid
def list_clients(conn):
    cur = conn.cursor()
    cur.execute("SELECT * FROM clientes")
    rows = cur.fetchall()
    for row in rows:
        print(row)
def update_client(conn, client):
    sql = ''' UPDATE clientes
              SET nombre = ? ,
                  apellidos = ? ,
                  email = ?
              WHERE Identificador = ?'''
    cur = conn.cursor()
    cur.execute(sql, client)
    conn.commit()
def delete_client(conn, id):
    sql = 'DELETE FROM clientes WHERE Identificador=?'
    cur = conn.cursor()
    cur.execute(sql, (id,))
    conn.commit()
def main():
    database = DB_NAME
    conn = create_connection(database)
    if conn is not None:
        create_table(conn)
        while True:
            print("Gestión de Clientes")
            print("1. Crear Cliente")
            print("2. Listar Clientes")
            print("3. Actualizar Cliente")
            print("4. Eliminar Cliente")
            print("5. Salir")
            choice = input("Selecciona una opción: ")
            if choice == '1':
                nombre = input("Nombre: ")
                apellidos = input("Apellidos: ")
                email = input("Email: ")
                client = (nombre, apellidos, email)
                create_client(conn, client)
                print("Cliente creado.")
            elif choice == '2':
                list_clients(conn)
            elif choice == '3':
                id = int(input("ID del cliente a actualizar: "))
                nombre = input("Nuevo Nombre: ")
                apellidos = input("Nuevos Apellidos: ")
                email = input("Nuevo Email: ")
                client = (nombre, apellidos, email, id)
                update_client(conn, client)
                print("Cliente actualizado.")
            elif choice == '4':
                id = int(input("ID del cliente a eliminar: "))
                delete_client(conn, id)
                print("Cliente eliminado.")
            elif choice == '5':
                break
            else:
                print("Opción no válida.")
```
