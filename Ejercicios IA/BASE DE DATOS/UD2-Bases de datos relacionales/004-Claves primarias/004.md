# Introducción: 
Este ejercicio es fundamental para comprender la importancia de las claves primarias en la gestión de bases de datos. Las claves primarias aseguran la integridad de los datos y facilitan la identificación única de cada registro, lo cual es esencial para operaciones como consultas, actualizaciones y eliminaciones. Además, el uso de auto-incremento simplifica la gestión de identificadores únicos, evitando conflictos y errores al insertar nuevos registros. 

Una clave primaria es un campo o conjunto de campos que identifican de manera única cada registro en una tabla. En este caso, se añade una columna `id_artista` como clave primaria a la tabla `artistas` en la base de datos `empresadam`.

---
**-- Conectar a la base de datos.**
```
sudo mysql -u root -p
```
**-- Seleccionar la base de datos.**
```
USE empresadam;
```
**-- Mostrar las tablas disponibles.**
```
SHOW TABLES;
```
**-- Describir la tabla artistas.**
```
DESCRIBE artistas;
```
**-- Se añade una clave primaria a la tabla artistas.**
```
ALTER TABLE artistas
ADD COLUMN id_artista INT AUTO_INCREMENT PRIMARY KEY FIRST;
```
**-- Se inserta un nuevo artista.**
```
INSERT INTO artistas (nombre, genero)
VALUES ('Extremoduro', 'Electronica');
```
**-- Verifica los datos insertados.**
```
SELECT * FROM artistas;
```
---
# Conclusión:
El ejercicio demuestra cómo añadir una clave primaria a una tabla existente en MySQL y cómo insertar un nuevo registro en esa tabla. La columna `id_artista` se ha configurado como una clave primaria con auto-incremento, lo que garantiza que cada artista tenga un identificador único.

Un error común al intentar añadir una clave primaria a una tabla que ya contiene datos es que los valores existentes en la columna que se desea convertir en clave primaria deben ser únicos y no nulos. Si hay duplicados o valores nulos, MySQL generará un error al intentar establecer la clave primaria. En este caso, al añadir una nueva columna `id_artista` con auto-incremento, se evita este problema, ya que MySQL asignará automáticamente valores únicos a cada fila existente.
