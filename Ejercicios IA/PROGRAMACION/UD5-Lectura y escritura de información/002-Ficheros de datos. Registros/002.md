# Introducción:
Este ejercicio ayuda a comprender cómo manejar archivos JSON para almacenar y recuperar datos estructurados, así como a aplicar estilos básicos en una página HTML para mejorar su presentación visual.

El archivo JSON contiene artículos de un blog sobre la banda de rock Extremoduro, y el código Python lee este archivo para mostrar su contenido en la consola. 
Además, se incluye un ejemplo de cómo aplicar estilos CSS básicos a una página HTML que presenta los artículos del blog.

A la hora de leer el archivo JSON, he tenido que especificar la ruta completa del archivo debido a problemas para detectarlo solo por su nombre pero normalmente no debería ser necesario puesto que el archivo se encuentra en la misma carpeta que el script de Python.

---
**Archivo JSON**
```
[
  {
    "titulo": "Extremoduro y la evolución del metal",
    "fecha": "2025-10-16",
    "autor": "Alfredo Martínez Cantero",
    "contenido": "Extremoduro es una rama del rock que combina elementos de rock y poesía."
  },
  {
    "titulo": "Las mejores canciones de Extremoduro",
    "fecha": "2025-10-17",
    "autor": "Alfredo Martínez Cantero",
    "contenido": "En este artículo presentaré algunas de las canciones más emblemáticas de Extremoduro."
  }
]
```
---
**Leemos el archivo JSON y mostramos su contenido:**
```
import json

ruta_blog = r"C:\Users\Usuario\Documents\DAM-2025\Ejercicios IA\PROGRAMACION\UD5-Lectura y escritura de información\002-Ficheros de datos. Registros\blog.json" ### Está esta ruta debido a que no me detectaba el archivo JSON por nombre.###

with open(ruta_blog, 'r', encoding='utf-8') as archivo:
    blog = json.load(archivo)

for articulo in blog:
    print("Título: " + articulo['titulo'])
    print("Fecha: " + articulo['fecha'])
    print("Autor: " + articulo['autor'])
    print("Contenido: " + articulo['contenido'])
    print("----------------------------------")
```
---
**Añadimos estilos básicos a una página HTML:**
```
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Blog de Extremoduro</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            line-height: 1.6;
            margin: 0;
            padding: 0;
        }
        header {
            background: #333;
            color: #fff;
            padding: 10px 0;
            text-align: center;
        }
        article {
            border-bottom: 1px solid #ccc;
            margin: 20px;
            padding: 20px;
        }
        h1, h2 {
            color: #333;
        }
</style>
</head>
<body>
    <h1>Mi Blog</h1>
    <h2>Extremoduro y la evolución del rock</h2>
    <p>Extremoduro es una rama del rock que combina elementos de rock y poesía.</p>

    <h2>Las mejores canciones de Extremoduro</h2>
    <p>En este artículo presentaré algunas de las canciones más emblemáticas de Extremoduro.</p>
</body>
</html>
```
