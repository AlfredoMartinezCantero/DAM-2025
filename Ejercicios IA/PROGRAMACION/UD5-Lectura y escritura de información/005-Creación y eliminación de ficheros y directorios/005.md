# Introducción:
Este ejercicio sirve para crear y eliminar archivos y directorios utilizando Python. Utilizaremos los módulos integrados `os` y `zipfile` para realizar estas operaciones.

`archivo = open("miarchivo.txt",'w')` crea un archivo de texto llamado `miarchivo.txt` en modo escritura. Si el archivo ya existe, se sobrescribirá. Si no existe, se creará uno nuevo.

`archivo.write("Esto es un texto de prueba que estoy escribiendo")` escribe la cadena de texto proporcionada en el archivo abierto.

`origen = 'crmca_crmcapitol (1).sql'` define la ruta del archivo que se desea comprimir.

`destino = 'basededatos.zip'` define el nombre del archivo ZIP que se creará.

`os.remove("miarchivo.txt")` elimina el archivo `miarchivo.txt` del sistema de archivos.

`os.mkdir("micarpeta")` crea un nuevo directorio llamado `micarpeta`

`for directorio, subcarpetas, archivos in os.walk(carpeta):` recorre todos los directorios y archivos dentro de la carpeta especificada.

`archivo = zipfile.ZipFile(destino, 'w', compression=zipfile.ZIP_DEFLATED)` crea un nuevo archivo ZIP con el nombre especificado y utiliza la compresión ZIP_DEFLATED para reducir el tamaño del archivo.

---
```
import zipfile
import os
```
---
**1.- Crear un archivo con texto.**
```
archivo = open("miarchivo.txt",'w')
archivo.write("Esto es un texto de prueba que estoy escribiendo")

archivo.close()
```
---
**2.- Comprimir un archivo.**
```
origen = 'crmca_crmcapitol (1).sql'
destino = 'basededatos.zip'

archivo = zipfile.ZipFile(destino, 'w')
archivo.write(origen)

archivo.close()
```
---
**3.- Eliminar un archivo.**
```
os.remove("miarchivo.txt")
```
---
**4.- Crear una carpeta.**
```
os.mkdir("micarpeta")
```
---
**5.- Comprimir todos los archivos de una carpeta.**
```
carpeta = 'archivos'
for directorio, subcarpetas, archivos in os.walk(carpeta):
    for nombre_archivo in archivos:
        origen = os.path.join(directorio, nombre_archivo)
        destino = os.path.join(directorio, nombre_archivo+ ".zip")
        archivo = zipfile.ZipFile(destino, 'w', compression=zipfile.ZIP_DEFLATED)
        archivo.write(origen, arcname=nombre_archivo)
        archivo.close()
```
---
