# Introducción:
Este ejercicio muestra cómo crear una clase en Python, instanciar objetos de esa clase, y guardar una lista de esos objetos en un archivo binario utilizando el módulo `pickle`.
Con lo que se puede aprender a manejar la serialización de objetos en Python y la gestión de archivos binarios.

`import pickle` es una biblioteca estándar de Python y se utiliza para serializar y deserializar objetos de Python, lo que permite guardar estructuras de datos complejas en archivos o transmitirlas a través de redes.

`clientes = [...]` crea una lista llamada `clientes` que contiene dos instancias de la clase `Cliente`, cada una inicializada con un nombre y un correo electrónico.

`with open("clientes.bin", "wb") as archivo:` abre un archivo llamado "clientes.bin" en modo de escritura binaria (`"wb"`). El uso de `with` asegura que el archivo se cierre automáticamente después de que se complete el bloque de código.

`pickle.dump(clientes, archivo)` serializa la lista `clientes` y la escribe en el archivo abierto. Esto permite guardar la estructura de datos completa en un formato que puede ser leído más tarde.

Por último, `print("Lista de clientes guardada en 'clientes.bin'")` muestra un mensaje en la consola indicando que la lista de clientes se ha guardado correctamente en el archivo binario.

---
```
import pickle

class Cliente:
    def __init__(self, nombre, email):
        self.nombre = nombre
        self.email = email

clientes = [
    Cliente("Alfredo Martínez", "alfredo@gmail.com"),
    Cliente("Rafa Cantero", "rafa@gmail.com")
]

with open("clientes.bin", "wb") as archivo:
    pickle.dump(clientes, archivo)

print("Lista de clientes guardada en 'clientes.bin'")
```

La capacidad de serializar objetos es fundamental en muchas aplicaciones, ya que permite almacenar y recuperar datos de manera eficiente y gracias a este ejercicio se hace más fácil de entender cómo manejar archivos binarios y a utilizar el módulo `pickle` para la persistencia de datos en Python.
