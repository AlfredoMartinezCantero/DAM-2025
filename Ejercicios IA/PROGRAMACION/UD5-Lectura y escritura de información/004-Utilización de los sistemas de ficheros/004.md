# Introducción:
Este ejercicio sirve para practicar la utilización de los sistemas de ficheros en Python. El objetivo es crear un programa que solicite al usuario una carpeta y calcule el tamaño total ocupado por los archivos en esa carpeta, así como el número de archivos que contiene.

`import os` es una librería estándar de Python que proporciona una forma de interactuar con el sistema operativo, incluyendo la manipulación de archivos y directorios.

`elementos = os.listdir(carpeta)` obtiene una lista de todos los elementos (archivos y subdirectorios) en la carpeta especificada por el usuario.

`os.path.join(carpeta, elemento)` se utiliza para construir la ruta completa a cada elemento en la carpeta.

`if os.path.isfile(ruta):` comprueba si la ruta corresponde a un archivo (y no a un directorio).

`suma += os.path.getsize(ruta)` obtiene el tamaño del archivo en bytes y lo añade a la variable `suma`.

---
```
import os

carpeta = input("Indica una carpeta: ")

elementos = os.listdir(carpeta)

suma = 0
contador_archivos = 0

for elemento in elementos:
    ruta = os.path.join(carpeta,elemento)
    if os.path.isfile(ruta):
        suma += os.path.getsize(ruta)
        contador_archivos += 1

print("La carpeta ocupa:")
print(suma/(1024*1024),"MB")
print("Número de archivos:", contador_archivos)
```
---
# Conclusión:
Este programa permite conocer el tamaño total ocupado por los archivos en una carpeta específica y el número de archivos que contiene. Utiliza funciones de la librería `os` para interactuar con el sistema de archivos, demostrando cómo listar elementos en un directorio, verificar si son archivos y obtener su tamaño. Es una herramienta útil para gestionar y analizar el uso del espacio en disco.
Un error común al ejecutar este programa es proporcionar una ruta de carpeta que no existe o a la que no se tiene acceso.
