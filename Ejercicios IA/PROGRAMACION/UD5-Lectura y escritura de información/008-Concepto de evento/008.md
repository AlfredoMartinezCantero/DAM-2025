# Introducción:
Este programa en Python utiliza la biblioteca Tkinter para crear una interfaz gráfica que muestra los datos de una tabla "clientes" de una base de datos MySQL en un widget Treeview. El programa se conecta a la base de datos, recupera todos los registros de la tabla "clientes" y los inserta en el Treeview para su visualización.

`arbol = ttk.Treeview(ventana, columns=("dninie","nombre", "apellidos","email"), show="headings")` crea un widget Treeview con columnas para mostrar el DNI, nombre, apellidos y email de los clientes. Luego, se ejecuta una consulta SQL para obtener todos los registros de la tabla "clientes" y se insertan en el Treeview utilizando un bucle. Finalmente, se muestra la ventana principal de la aplicación.

`for fila in filas:` itera sobre cada fila obtenida de la consulta SQL y `arbol.insert("", "end", values=(fila[1], fila[2], fila[3], fila[4]))` inserta cada registro en el Treeview, mostrando los datos correspondientes en las columnas definidas.

---
```
import tkinter as tk
from tkinter import ttk
import mysql.connector
conexion = mysql.connector.connect(host="localhost",user="empresadam",password="Empresadam123$",database="empresadam")
cursor = conexion.cursor()
ventana = tk.Tk()
arbol = ttk.Treeview(ventana, columns=("dninie","nombre", "apellidos","email"), show="headings")
arbol.heading("dninie", text="DNI del cliente")
arbol.heading("nombre", text="Nombre del cliente")
arbol.heading("apellidos", text="Apellidos del cliente")
arbol.heading("email", text="Email del cliente")
cursor.execute('''SELECT * FROM clientes;''')
filas = cursor.fetchall()
for fila in filas:
  arbol.insert("", "end", values=(fila[1], fila[2], fila[3], fila[4]))

arbol.pack(padx=20,pady=20)

ventana.mainloop()
```

# Conclusión:
Este programa demuestra cómo utilizar Tkinter para crear una interfaz gráfica que interactúa con una base de datos MySQL. Al emplear el widget Treeview, se facilita la visualización de datos tabulares de manera organizada y accesible. 

La conexión a la base de datos y la recuperación de datos mediante consultas SQL permiten integrar funcionalidades de backend con una interfaz de usuario amigable. Este enfoque es útil para aplicaciones que requieren mostrar información dinámica almacenada en bases de datos.

Un error común al escribir el programa podría ser equivocarse en los nombres de las columnas al definir el Treeview o al insertar datos, lo que resultaría en una visualización incorrecta o en errores de ejecución. Además, es importante asegurarse de que la conexión a la base de datos se establezca correctamente y que la tabla "clientes" exista con los campos esperados.
