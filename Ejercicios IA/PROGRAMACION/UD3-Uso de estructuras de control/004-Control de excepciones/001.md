# Introducción:
Este ejercicio nos enseña a manejar excepciones en Python utilizando bloques try y except. 
Gracias a esto aprendemos a capturar errores durante una simulación de conexión a una base de datos y a proporcionar mensajes amigables al usuario.
Esto lo conseguimos mediante la creación de una función llamada `conectar_base_de_datos()` que intenta realizar una operación que genera un error (división por cero) y maneja ese error adecuadamente.

---
```
def conectar_base_de_datos():
    print("")
    print("Intentando conectar a la base de datos...")
    print("")
    try:
        resultado = 10 / 0
        print("Hemos conectado con la base de datos!")
    except:
        print("Ooops... Ha ocurrido un error al conectarse a la base de datos.")
        print("")
        print("Por favor, verifica tu conexión a internet y vuelve a intentarlo.")

conectar_base_de_datos()
print("")
print("El programa sigue ejecutándose...")
print("")
```

# Conclusión:
Este ejercicio es fundamental para entender cómo manejar errores en Python, ya que el uso de bloques try y except nos permite capturar excepciones y proporcionar una experiencia de usuario más amigable, evitando que el programa se detenga de repente ante errores inesperados.
Al practicar este tipo de manejo de excepciones, mejoramos la robustez y la fiabilidad de nuestras aplicaciones.
