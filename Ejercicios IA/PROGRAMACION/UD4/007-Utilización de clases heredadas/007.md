# Introducción:
Este ejercicio consiste en crear una jerarquía de clases para representar animales. La clase base se llamará `Animal` y tendrá atributos como `edad`, `nombre` y `raza`. Luego, se crearán dos clases derivadas: `Gato` y `Perro`, que heredarán de la clase `Animal`.
Este ejercicio nos ayudará a entender cómo funciona la herencia en la programación orientada a objetos.

El ejercicio comienza con la definición de la clase madre `Animal`, que tiene un constructor para inicializar los atributos `edad`, `nombre` y `raza`. 

A continuación, se definen las clases derivadas `Gato` y `Perro`, que llaman al constructor de la clase base utilizando `super()`. 

Por último, se crean instancias de `Gato` y `Perro`, y se imprimen sus atributos.

---
```
class Animal():
    def __init__(self,edad,nombre,raza):
        self.edad = edad
        self.nombre = nombre
        self.raza = raza

class Gato(Animal):
    def __init__(self,edad,nombre,raza):
        super().__init__(edad,nombre,raza)

class Perro(Animal):
    def __init__(self,edad,nombre,raza):
        super().__init__(edad,nombre,raza)

gato1 = Gato(2, "Pascal", "Siames")
print("El gato se llama", gato1.nombre, "y tiene", gato1.edad, "años")

perro1 = Perro(4, "Madacascar", "Golden retriever")
print("El perro se llama", perro1.nombre, "y tiene", perro1.edad, "años")
```
---
# Conclusión:
Gracias a este ejercicio se puede aprender a crear una jerarquía de clases utilizando la herencia en Python. La clase base `Animal` permite definir atributos comunes para todos los animales, mientras que las clases derivadas `Gato` y `Perro` heredan estos atributos y pueden ser instanciadas con sus propios valores específicos. Esto demuestra cómo la herencia facilita la reutilización del código y la organización de las clases en una estructura lógica.
