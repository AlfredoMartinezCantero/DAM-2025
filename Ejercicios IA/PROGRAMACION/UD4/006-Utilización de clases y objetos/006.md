# Introducción:
Este ejercicio consiste en crear una clase llamada `Matematicas` que contenga métodos para realizar operaciones matemáticas básicas como redondeo, techo y suelo de un número decimal. Además, la clase debe tener un atributo que almacene el valor de PI.

Gracias a este ejercicio, se puede practicar la creación y uso de clases y objetos en Python, así como la implementación de métodos para realizar cálculos matemáticos.

`def __init__(self):` es el método constructor de la clase, que se ejecuta automáticamente al crear una instancia de la clase. En este caso, se utiliza para inicializar el atributo `PI`.

`def redondeo(self, numero):` es un método que toma un número decimal como argumento y devuelve su valor redondeado al entero más cercano.

`def techo(self, numero):` es un método que toma un número decimal como argumento y devuelve el entero **más pequeño** que es mayor o igual al número dado.

`def suelo(self, numero):` es un método que toma un número decimal como argumento y devuelve el entero **más grande** que es menor o igual al número dado.

---
```
class Matematicas():
    def __init__(self):
        self.PI = 3.14159265359
    
    def redondeo(self, numero):
        entero = int(numero)
        decimal = numero - entero
        if decimal < 0.5:
            redondeo = 0
        else:
            redondeo = 1
        return entero + redondeo

    def techo(self, numero):
        entero = int(numero)
        decimal = numero - entero
        if decimal > 0:
            techo = 1
        else:
            techo = 0
        return entero + techo

    def suelo(self, numero):
        entero = int(numero)
        return entero
    
Mate = Matematicas()
print("Redondeo de 4.7:", Mate.redondeo(4.7))
print("Redondeo de 4.2:", Mate.redondeo(4.2))
print("Valor al techo de 4.7:", Mate.techo(4.7))
print("Valor al suelo de 4.7:", Mate.suelo(4.7))
```
