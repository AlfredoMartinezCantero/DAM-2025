# Introducción:
Este ejercicio sirve para practicar la creación de constructores en Python mediante la definición de clases. Un constructor es un método especial que se llama automáticamente cuando se crea una nueva instancia de una clase. En Python, el constructor se define utilizando el método `__init__`.

En el primer ejercicio, se define una clase `Gato` con un constructor que inicializa los atributos `nombre` y `edad` con valores predeterminados.

En el segundo ejercicio, se modifica el constructor para aceptar un parámetro que permita establecer la edad del gato al crear una instancia.

Finalmente, en el tercer ejercicio, se crea una clase `Cliente` con un constructor que acepte varios parámetros para inicializar los atributos del cliente.

---
**Ejercicio 1:**
```
class Gato():
    def __init__(self):
        self.nombre = "Misifu"
        self.edad = 0

    def maulla(self):
        return "está maullando"

gato1 = Gato()
print("El gato se llama", gato1.nombre)
print("")
print(gato1.nombre, "tiene", gato1.edad, "años")
print("")
print("Parece que", gato1.nombre, gato1.maulla())
print("")
```

**Ejercicio 2:**
```
class Gato():
    def __init__(self,edad):
        self.nombre = "Pascal"
        self.edad = edad

    def maulla(self):
        return "está maullando"

gato2 = Gato(5)

print("")

print("")
```

**Ejercicio 3:**
```
class Cliente():
    def __init__(self,nombre,apellidos,email,direccion):
        self.nombre = nombre
        self.apellidos = apellidos
        self.email = email
        self.direccion = direccion

nombre = input("Introduce el nombre del cliente: ")
print("")
apellidos = input("Introduce los apellidos del cliente: ")
print("")
email = input("Introduce el email del cliente: ")
print("")
direccion = input("Introduce la dirección del cliente: ")
print("")

cliente1 = Cliente(nombre,apellidos,email,direccion)
print("-----------------------------------------------------------------")
print("Nombre: ", cliente1.nombre)
print("")
print("Apellidos: ", cliente1.apellidos)
print("")
print("Email: ", cliente1.email)
print("")
print("Dirección: ", cliente1.direccion)
print("")
```

# Conclusión:
Este ejercicio, sirve para crear constructores en Python utilizando el método `__init__`. Gracias a este, se puede ver cómo inicializar atributos con valores predeterminados y cómo aceptar parámetros para personalizar la creación de instancias de una clase. Estos conceptos son fundamentales para trabajar con programación orientada a objetos en Python.

Un error común al crear constructores es olvidar el parámetro `self`, que es esencial para referirse a los atributos y métodos de la instancia actual. Asegurarse de incluir `self` en la definición del constructor y en otros métodos de la clase es crucial para evitar errores y garantizar que el código funcione correctamente.
